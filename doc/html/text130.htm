Universal Map





UniversalMap(C, S, [ n1, ..., nm ]) : Cop, Str, [ Map ] -> Map



Given maps n1, ..., nm from structures S1, ..., Sm
that compose the coproduct C, to some structure S, this function returns the
universal map C -> S.
