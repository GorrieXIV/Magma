Introduction






Functions are one of the most fundamental elements of the Magma language.
The first section describes the various ways in which a standard function may 
be defined while the second section describes the definition of a procedure 
(i.e. a function which doesn't return a value). The second half of the chapter 
is concerned with user-defined intrinsic functions and procedures.
