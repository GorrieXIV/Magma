Roots of Elements










Roots of local ring and field elements can be found to some precision.

results if x is not a square.  The result may have less precision than x.

result may have less precision than x.

InverseSqrt(x, y) : FldPadElt, FldPadElt -> FldPadElt



Given local ring or field elements x and y, return an inverse square root
of x lifted from an initial approximation y.  The element x must be a
unit.  An error results if x is not a square, or if y is not a valid
initial approximation to an inverse square root of x.  The result may have
less precision than x.
