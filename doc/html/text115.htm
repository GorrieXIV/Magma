Construction of Lists






Lists can be constructed by expressions enclosed in special brackets
[*  and *].



[*  *] : -> List



The empty list.



[*  e1, e2, ..., en *] : Elt,.., Elt -> List



Given a list of expressions e1, ..., en, defining elements 
a1, a2, ..., an, create the list containing
a1, a2, ..., an.
