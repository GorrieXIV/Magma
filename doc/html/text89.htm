Sets from Structures





Set(M) : Str -> SetEnum



Given a finite structure that allows explicit enumeration
of its elements, return the set containing its elements
(having M as its universe).



FormalSet(M) : Str -> SetFormal



Given a structure M, return the formal set consisting
of its elements.
