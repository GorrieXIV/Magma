Creating Names






Sometimes it is necessary to create names for files from within Magma
that will not clash with the names of existing files.



Tempname(P) : MonStgElt -> MonStgElt



Given a prefix string P, return a unique temporary name derived from P
(by use of the C library function mktemp()).
