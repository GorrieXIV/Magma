Moving around the HTML Help Document

The screens of the Document are arranged in a tree.  On every screen,
there are buttons at the top of the screen (and sometimes at the bottom also)
which allow you to move around the tree.
Here is a brief explanation of the buttons.



[Next]


    Move to the next screen in a depth-first traversal of the tree.  That is,
    move down if possible; otherwise move right if possible; otherwise keep
    moving up until moving right is possible and then move right.




[Prev]


    Move to the previous screen in a depth-first traversal of the tree.
    That is, move to the screen from which the [Next] button leads to this
    screen.




[Right]


    Move to the screen immediately right to this one if possible (that is,
    the next section, chapter, or part, etc.---useful for skipping subscreens
    you're not interested in while browsing).




[Left]


    Move to the screen immediately left to this one if possible.




[Up]


    Move to the screen immediately above this one (useful for moving back
    to the enclosing chapter or part, etc.).




[Index]


    Move to the Index screen.  The Index screen leads to other Index screens
    which contain references to sections and Magma Intrinsics discussed
    in the Document.




[Root]


    Move to the root of the tree (this screen).
