Automorphism Groups





AutomorphismGroup(G) : GrpAb -> GrpAuto



The full automorphism group of the abelian group G.



IsIsomorphic(G, H) : GrpAb, GrpAb -> BoolElt, Hom(Grp)



Test whether or not the two abelian groups G and H are
isomorphic as abstract groups. If so, both the result true and
an isomorphism from G to H is returned. If not, the result false
is returned.
