Memory Usage





GetMemoryUsage() : -> RngIntElt



Return the current memory usage of Magma (in bytes as an integer).
This is the process data size, which does not include the executable code.



GetMaximumMemoryUsage() : -> RngIntElt



Return the maximum memory usage of Magma (in bytes as an integer)
which has been attained since last reset (see ResetMaximumMemoryUsage).
This is the maximum process data size, which does not include the
executable code.



ResetMaximumMemoryUsage() : ->



Reset the value of the maximum memory usage of Magma to be the current
memory usage of Magma (see GetMaximumMemoryUsage).
