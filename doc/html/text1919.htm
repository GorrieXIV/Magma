LPProcess(R, n) : Rng, RngIntElt -> LP



A Linear Program over the ring R in n variables.





Example LP_LPCreation (H166E3)

We create an LP representing a problem in 2 real variables:





> R := RealField( );          
> L := LPProcess(R, 2);
> L;
