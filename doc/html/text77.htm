Introduction






In ordered to facilitate the debugging of complex pieces of Magma code,
Magma includes a debugger.  This debugger is very much a prototype, and
can cause Magma to crash.



SetDebugOnError(f) : BoolElt ->



If f is true, then upon an error Magma will break into the debugger.  The
usage of the debugger is described in the next section.
