Introduction






An associative array in Magma is an array which may be
indexed by arbitrary elements of an index structure I.  The indexing
may thus be by objects which are not integers.  These objects are
known as the keys.  For each current key there is an associated
value.  The values associated with the
keys need not lie in a fixed universe but may be of any type.
