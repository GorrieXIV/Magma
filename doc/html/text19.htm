IsIntrinsic(S) : MonStgElt -> Bool, Intrinsic



Given a string S, return true if and only an intrinsic with the name
S exists in the current version of Magma.  If the result is true,
return also the actual intrinsic.





Example State_IsIntrinsic (H1E23)

We demonstrate the function IsIntrinsic.





> IsIntrinsic("ABCD");
false
> l, a := IsIntrinsic("Abs");
> l;
true
> a(-3);
3
