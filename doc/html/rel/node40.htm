Permutation Groups


New Features:

  • Structural algorithms for permutation groups that depend upon computing the socle have been extended to apply to all permutation groups. The theory behind the socle algorithm for a primitive group has been extended to degree 232 , which exceeds the current degree limit for a permutation group of 230 - 1 . The algorithm for imprimitive groups has also been improved. The algorithms affected include those for socle, chief series, simplicity tests, composition series, and composition factors. (V2.21-4)

  • The IsAltsym routine now uses an improved probabilistic test for a transitive group to contain the alternating group. At high degree, this test uses many fewer random group elements than the previous test.
