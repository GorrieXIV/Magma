Basic Algebras


New Features:

  • Intrinsics Quiver and QuiverAndRelations construct the quiver and its relations for a basic algebra.

  • The new intrinsic InnerAutomorphismGroup constructs the group of inner automorphisms for a basic algebra.

  • A small library of basic algebras for the p -modular group algebras of some of the smaller Atlas groups has been constructed by Jon Carlson. This library may be accessed using the intrinsics BasicAlgebraFromGroup and BasicAlgebraGroupNames.

  • A second library contains a collection of basic algebras for Schur algebras and can be accessed using the intrinsic BasicAlgebraFromSchur.
