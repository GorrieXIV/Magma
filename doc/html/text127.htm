Accessing Functions





Injections(C) : Cop -> [ Map ]



Given a coproduct C, return the sequence of injection maps returned
as the second argument from the cop<> constructor.



# C : Cop -> RngIntElt



Given a coproduct C, return the length (number of constituents) of C.



Constituent(C, i) : Cop, RngIntElt -> Str



Given a coproduct C and an integer i between 1 and the length of C,
return the i-th constituent of C.



Index(x) : CopElt -> RngIntElt



Given an element x from a coproduct C, return the constituent number
of C to which x belongs.
