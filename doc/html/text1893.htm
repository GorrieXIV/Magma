Automorphism Group





AutomorphismGroup(C) : CodeAdd -> GrpPerm



The automorphism group of the additive code C.  Currently, this 
function is only available for additive codes over GF(4).



PermutationGroup(C) : CodeAdd -> GrpPerm



The subgroup of the automorphism group of the additive code C
consisting of permutations of the coordinates. Currently, this 
function is only available for additive codes over GF(4).
