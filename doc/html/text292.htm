Introduction










This chapter describes Magma functions for working with quadratic, bilinear 


and sesqui-linear forms defined on vector spaces.  The emphasis is on vector 
spaces defined over finite fields but in some instances the functions apply 
more widely.  For quadratic forms defined on lattices see Chapter 
QUADRATIC FORMS. For the interpretation of reflexive forms as algebras
with involution see Chapter ALGEBRAS WITH INVOLUTION. General references for this 
material are [Bou07], [Tay92].
