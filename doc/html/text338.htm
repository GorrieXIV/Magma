Introduction






This chapter describes miscellaneous functionality for fairly general
quadratic forms. The main feature currently here is an implementation of 
Simon's algorithm for finding isotropic subspaces of integral forms. 


In Magma, quadratic forms are generally represented either as
multivariate polynomials, or as symmetric matrices.
